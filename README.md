## 🌟 What Is This?

I designed a **custom FPGA development board** using Lattice **MachXO2** FPGA in KiCad.

This is **hardware PCB design** (not code) - I designed the board itself:

- MachXO2 FPGA chip (LCMXO2-7000HE)
- Power supply 5V to 3.3V
- 12MHz crystal clock
- JTAG header for programming
- User LEDs and Button
- GPIO headers

**Why I made my own board?** To learn PCB design for FPGA and have my own hardware.

## ✨ Board Features

- ✅ Lattice MachXO2 FPGA - LCMXO2-7000HE-4-TG144C
- ✅ 12MHz crystal oscillator
- ✅ 3.3V LDO regulator
- ✅ JTAG header for programming
- ✅ 2 User LEDs
- ✅ 1 User Button
- ✅ GPIO headers
- ✅ Power LED
- ✅ 2-layer PCB

## 🛠️ Built With

- **Tool:** KiCad 7.0
- **FPGA:** Lattice MachXO2 LCMXO2-7000HE
- **PCB:** 2-layer
